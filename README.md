# Stock News Sentiment Analysis

A lightweight project for analyzing financial news sentiment and relating it to stock market trends. This project uses a custom API template (see https://github.com/Niilop/My-FastAPI-template) as the backend foundation.

# Overview

This project fetches stock-related news articles and applies sentiment analysis to determine whether the overall tone is positive, negative, or neutral on user input stocks. The goal is to provide insights that could help inform trading decisions or market research.
